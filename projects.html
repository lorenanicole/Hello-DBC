HTML & CSS - Building Simple Sites

Following a Treehouse tutorial, I built a HTML & CSS website for a fictionalized bakery "Smells Like Bakin'" viewable here. Some quick take aways I had from this exercise:

• Using grids is a simple way to build a clean, professional style for a website

• Normalizing CSS will optimize cross-browser compatibility

• Everything you need to know about CSS properties is always a web search away (hint: Check W3!)

Geeks Without Bounds: Accelerating Red Cross Disaster Response Time

After an intense Hackathon spanning an entire weekend in June (quaintly nammed 'Everyone Hacks Hack athon'), me and some eight other civic minded do-gooders were invited to participate in the Geeks Without Bounds Accelerator. Our project? To oversee the implementation of ARCDAT (American Red Cross Disaster Action Team created by Bay Area Developer John Laxson) -- a tool for dispatching and responding to diasters -- in the Greater Chicago Red Cross Chapter.

The Accelerator provides us with meaningful and relevant mentorships and technology support as we continue to develop and roll out the implementation of our application. All chats are recorded, like you see below, and made accessible for anyone wishing to develop likeminded humanitarian applications.
